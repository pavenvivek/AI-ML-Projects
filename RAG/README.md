# AI/ML Projects
This repository contains the code for retrieval augmented generation (RAG). 

RAG needs two language model architectures: causal LLM for generation and sentence transformer for retrieval. 

The code in this repository uses pre-trained LLMs from kerashub and huggingface. It also contains the framework for locally training causal LLM and sentence transformer, and using them instead.
